    Kerloud Telemetry¶

    Kerloud Telemetry is a reliable short-range telemetry presented by Cloudkernel Technologies (Shenzhen) Co.,Ltd for unmanned systems. The product is designed based on ISM Sub-G frequency domain, and featured with FSK modulation and high anti-disturbance performance. It can be connected with Pixhawk/Ardupilot autopilots easily with UART/USB interfaces.

    Specifications¶

    Specification Parameter Remark
    Working frequency 915MHz
    Power 100mW（20dBm）
    Baud rate 57600（default）
    Transmission range 800m @ 64kbps Outdoor empty environment
    Interface Uart（GH1.25 4Pin）/Micro USB
    Input voltage 3.8~5.5V @ Uart, 4.75~5.25V @ USB
    3.3V

    Current during receiving 25mA Typical value
    Working temperature -20~85 ℃
    Size (without antenna) 55（Length）×30（Width）×7（Height） unit：mm
    Weight (with antenna) 15 unit：g

    Features¶

    • Full ESD protection
    • High bandwidth CMOS analog switch channel
    • Professional antenna matching with proved wireless performance
    • Uniform interface, able to work either onboard or with ground control station
    • Configurable wireless channels, allowing multiple modules to work simultaneously
    • LED Indication
