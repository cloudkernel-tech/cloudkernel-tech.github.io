# Kerloud Uno 自驾仪

<p align="center">
<img src ="../../assets/kerloud_uno/uno_view.jpg" width = 500/>
</p>

Kerloud Uno 是云讷科技于2023年推出的第二代自动驾驶仪，它在发布时符合最新的 pixhawk FMUv6x 设计标准，其设计显著提高了计算能力和外设接口。

## 产品特性

(1) 升级至Arm Cortex-M7 MCU （STM32H743IIK6），最大CPU频率为480MHz。

(2) 采用微型设计，在市场上现有的 pixhawk FMUv6x 自动驾驶仪中尺寸最小。

(3) 具有 FMU 和 IO 端口的扩展接口，支持具有 I2C、SPI、CAN 总线的各种外设。

(4) 全面的ESD（静电放电）保护，用于提高安全质量的接口。

(5) LTS（长期支持）承诺长达 5 年，以确保硬件和软件的稳定性。

## 指标

| 部件 | 指标 |
| :--: | :--: |
| 主处理器 |   STM32H743IIK6 Cortex M7, 2MB flash, maximum 480MHz cpu frequency |
| 协处理器 |   STM32F100 Cortex M3, 64KB flash, 24MHz cpu frequency |
| 加速度计 |   ICM42670-P, BMI088 |
| 陀螺仪 |  ICM42670-P, BMI088 |
| 磁感计 |  IST8310 (外置), BMM150 (内置) |
| 气压计 |  BMP388 |
| Kerloud GPS |  Ublox M8N GPS，外置磁感计IST8310 |
| 供电电压 |  Power port: 4.75V~5.5V, <br> USB port: 4.75V~5.25V, <br> 自带分电板: 2~8S, <br> 最大传感电流 90A, <br> BEC: 5.3V, 最大连续电流 3A |
| 工作温度范围 | -20 ~ 60 度|
| 外壳| CNC 铝制外壳 |
| 尺寸 |  57mm * 45mm * 18.7mm |
| 重量 |  Kerloud Uno Autopilot: 50g, Kerloud GPS: 30g |
