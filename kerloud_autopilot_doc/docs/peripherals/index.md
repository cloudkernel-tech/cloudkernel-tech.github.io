# Peripherals

We provide the following peripherals for Kerloud autopilots:

- [Kerloud Lite power module](kerloud_lite.md): a unit that can power up commonly used payloads in a drone.

- [Kerloud telemetry](kerloud_telemetry.md): a cost-effective telemetry for short range communication between Kerloud autopilots
and a ground control station.
