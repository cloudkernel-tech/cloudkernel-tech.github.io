# Kerloud Telemetry

Kerloud Telemetry is a reliable short-range telemetry presented by Cloudkernel Technologies (Shenzhen) Co.,Ltd for unmanned
systems. The product is designed based on ISM Sub-G frequency domain, and featured with FSK modulation and high anti-disturbance performance.
It can be connected with Pixhawk/Ardupilot autopilots easily with UART/USB interfaces.

## Features

* Full ESD protection
* High bandwidth CMOS analog switch channel
* Professional antenna matching with proved wireless performance
* Uniform interface, able to work either onboard or with ground control station
* Configurable wireless channels, allowing multiple modules to work simultaneously
* LED Indication
