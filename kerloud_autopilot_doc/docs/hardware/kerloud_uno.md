# Kerloud Uno Autopilot

Kerloud Uno is the 2nd generation autopilot presented by Cloudkernel Technologies in 2023. 
It complies with the latest pixhawk FMUv6x design standard at the release moment. The Kerloud Uno 
brings significant improvements in computation power and peripheral interfacing.

## Key Features

(1) Upgraded to Arm Cortex-M7 MCU (STM32H743IIK6) with maximum 480MHz CPU frequency.

(2) Miniature design with the smallest size among existing pixhawk FMUv6x autopilots in market.

(3) Extended interfaces with both FMU and IO ports, supporting various peripherals with I2C, SPI, CAN buses.

(4) Full ESD（Electro-Static Discharge）protection for interfaces with improved safety quality.

(5) LTS (Long term support) promise up to 5 years to ensure stability in both hardware and software.


## Specifications

| Component | Specification |
| :--: | :--: |
| Main Processor |   STM32H743IIK6 Cortex M7, 2MB flash, maximum 480MHz cpu frequency |
| Coprocessor |   STM32F100 Cortex M3, 64KB flash, 24MHz |
| Accelerometer |   ICM42670-P, BMI088 |
| Gyroscope |  ICM42670-P, BMI088 |
| Magnetometer |  IST8310 (external), BMM150 (Onboard) |
| Barometer |  BMP388 |
| Kerloud GPS Module |  Ublox M8N GPS,with external magnetometer IST8310 |
| Supply Voltage |  Power port: 4.75V~5.5V, <br> USB port: 4.75V~5.25V, <br> default power module: 2~8S, <br> maximum sensing current 90A, <br> BEC: 5.3V, maximum continuous current 3A |
| Operating Temperature | -20 ~ 60 degrees|
| Enclosure| CNC Aluminium enclosure |
| Dimension | 57mm * 45mm * 18.7mm |
| Weight | Kerloud Uno Autopilot: 50g, Kerloud GPS: 30g |
